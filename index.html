iOS Programming Projects

This web page was created to showcase a few of the programming projects that I have been working on.

GroupUp

  • Prototyped an app to find and form local groups using Core Location, MapKit, and Firebase
  • Used Core Animation to animate the custom views to pop up and shrink down in a clean manner
  • Added a slide-in menu using UIPanGestureRecognizers and UITapGestureRecognizers to animate the opening and closing of the left menu as the user slides their finger
  • Downloaded photos from Firebase Strorage using SDWebImage to cache the images and minimize network usage
  • Created custom UITableViewCells to display the group name, group photo, and the distance to the user at a glance
  • Constrained the size and placement of all UIViews using AutoLayout to ensure a clean presentation on all devices

Set Card Game

This is an iOS version of the card game Set. This project involved: combinatorics (Pascal's rule), Gesture recognizers, Core Animation, drawing custom UIViews using UIBezierPaths and CGAffineTransforms. Originally a homework assignment for CS193p at Stanford University. Grid.swift file provided by the class.
