GroupUp

This is an app to find and form local groups. This projects involves: Core Data, Mapkit, Core Location, Unit Testing, Geolocation formulas, TableViews, and Data Abstraction.

To do: create custom group annotation views, create a group info page, create group chats using Chatto, convert tableview to collection view using custom views.

Set Card Game

This is an iOS version of the card game Set. This project involved: combinatorics (Pascal's rule), Gesture recognizers, Core Animation, drawing custom UIViews using UIBezierPaths and CGAffineTransforms. Originally a homework assignment for CS193p at Stanford University. Grid.swift file provided by the class.

To do: animate score changes, add a matched cards view controller to display sets already matched. Source Code.
